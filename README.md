# pyType
Simple Typing mod for python

usage:
put this at the top to import the module:
import pyType.pyType as pt

use it as 
> pt.pType_d("hello")

or

> pt.iType_d("hello")

Prefixes:

>prefix p for print
>
>.prefix i for inputs


Speed (change from _d)

>_s for slow
>
>_ss for super slow

>_f for fast
>
>_sf for super fast


put the pyType.py file in a folder called pyType in the directory of the python file
