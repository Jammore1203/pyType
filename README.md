# pyType
Simple Typing mod for python
<br>
usage:
put this at the top to import the module:
import pyType.pyType as pt
<br>
use it as 
pt.pType_d("hello")
or
pt.iType_d("hello")
<br>
prefix p for print
prefix i for inputs
<br>
change _d for
_s for slow
_ss for super slow
<br>
_f for fast
_sf for super fast
